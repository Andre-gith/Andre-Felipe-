# 🚀 Portfólio — André Felipe

Portfólio pessoal desenvolvido do zero com HTML, CSS e JavaScript puro. Design responsivo com tema escuro, animações e seções de projetos, habilidades e contato.

**🔗 [Acesse o site](https://andrefelipe.netlify.app)**

![Preview do portfólio](./assets/preview.png)

## ✨ Funcionalidades

- Animação de digitação com **Typed.js** exibindo as stacks
- Menu mobile responsivo com animação hamburger
- Seção **Sobre** com skills e estatísticas
- Seção **Projetos** com cards linkados ao GitHub
- Seção **Contato** com WhatsApp, e-mail, LinkedIn e GitHub
- Scroll suave entre seções com destaque ativo no menu
- Link direto para download do currículo em PDF
